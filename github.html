Git og Github

Utført: 1 Oktober 2018

Git er et system for å følge med på forandringer i filer og for å dele disse filene mellom flere personer. Git er laget av Linus Torvalds i 2005. Linus Torvalds lagde også Linux.

GitHub er en nett-basert host for source code. GitHub gir deg muligheten til å ha private prosjekter(koster penger) og open-source prosjekter(gratis). GitHub har 28 millioner brukere og 57 millioner prosjekter. GitHub ble nylig kjøpt opp av Microsoft for 61 milliarder kroner.

Branching er en av de største fordelene med Git. I Git er det billigere(raskt og bruker lite plass) og lettere å slå sammen enn vanlige versjonskontrollsystemer. Git er fleksibelt, raskt og gratis. Jeg synes at Git har et bra konsept. Problemet jeg har med Git er at læringskurven er bratt. Det er ikke alltid like lett å huske alle kommandoene. Ellers er jeg fornøyd med Git.
