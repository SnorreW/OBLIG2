Media typer

Utført: 1 Oktober 2018

  • All: Passer for alle enheter.
  • Braille: Passer for blindeskrift enheter.
  • Embossed: Passer for blindeskrift printere.
  • Handheld: Passer for håndholdte. Blir nesten ikke brukt lenger på grunn av responsivt design.
  • Print: Passer for sider og dokumenter sett på i print preview mode. Blir brukt til utskrift. Sideoppsett og -brytning er den største forskjellen mellom Print og Skjerm.
  • Projection: Passer for presentasjoner med prosjektorer. Blir nesten ikke brukt lenger på grunn av responsivt design.
  • Screen: Passer for dataskjermer med farge. Vi bruker det det når vi ser på nettsider med en laptop, nettbrett, telefon osv. Man kan si det er det "vanlige" formatet. Det er nyttig for å skille mellom stilark som er tilpasset ulike medietyper.
  • Speech: Passer for opplesning av elementer.
  • Tty: Passer for terminaler og enheter med begrenset display muligheter
  • Tv: Passer for fjernsyns-type enheter.

Med print og speech kan man gi ut en bok i HTML og CSS som kan printes ut og gis ut som lydbok. Andre medietyper dekkes av regler for tilgjengelighet og responsivt design.
