CSS Layouts

Utført: 1 Oktober 2018

CSS Grid er et 2-dimensjonelt layout-system som bruker både kolloner og rader. Grid vil forrandre måten vi jobber med rutenett-design i nettsider. Man starter med å definere en container som grid ved display:grid, så setter man child-elementer inn i containeren. Samme som flexbox, så har rekkefølgen på elementene ikke noe å si. Dette gjør det lettere å gjøre om rutenettet med media queries. Grid er en av de sterkeste CSS modulene som er gitt ut. De fleste browsere støtter Grid. Chrome, Firefox, Safari og Opera støtter grid. Internet Explorer 10 og 11 støtter grid men syntaxen er utdatert.

CSS Flexbox gir en effektiv løsning på å lage rom mellom elementer i en container, selv når bredden ikke er gitt. Derfor navnet "flex". Hovedideen med flexbox er å fylle ut ledig rom på best mulig måte. En flex container kan utvide seg for å fylle ledig rom, eller så blir den mindre for å unngå overlapping. Ulikt Grid, så følger flexbox kun et 1-dimensjonalt system(horisontalt).
